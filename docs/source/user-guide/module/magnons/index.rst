*****************
Magnon dispersion
*****************

For the full reference see :ref:`api_magnons`.

.. currentmodule:: radtools

This module handles calculation of magnon dispersion. For the method see
:ref:`library_magnon-dispersion-method`.

Most of the functionality is implemented in the :class:`MagnonDispersion` class.
For the guide see

.. toctree::
   :maxdepth: 1
   :caption: Individual guide on

   dispersion

Computation of magnon dispersion involves diagonalization of the bosonic Hamiltonian.
This procedure is implemented as a separate function and can be used for digitalisation
of arbitrary bosonic Hamiltonian. For the guide see
