*********
Collinear
*********

Command-line
============

Default style
-------------

.. code-block:: bash

    rad-plot-dos.py -if collinear/ -ew -6.5 6.5 -ef -1.7806

Relative style, normalized
--------------------------

.. code-block:: bash

    rad-plot-dos.py -if collinear/ -ew -6.5 6.5 -ef -1.7806  -r -n
