*************************
Non-collinear, spin-orbit
*************************

Command-line
============

Default style
-------------

.. code-block:: bash

    rad-plot-dos -if noncollinear-so/ -ew -6.5 6.5 -ef -1.6372

Relative style, normalized
--------------------------

.. code-block:: bash

    rad-plot-dos -if noncollinear-so/ -ew -6.5 6.5 -ef -1.6372  -r -n
